How to Export

MT5

Open MT5 → View → Terminal → Account History tab → Right-click → Save as Report (HTML) → then convert to CSV, or use the "Export to CSV" option if available. Required columns: Ticket, Open Time, Type, Size, Symbol, Open Price, Close Time, Close Price, Commission, Swap, Profit.

cTrader

Open cTrader → History → Positions tab → Right-click → Export CSV. Required columns: Position ID, Symbol, Direction, Volume (Lots), Open Time, Open Price, Close Time, Close Price, Commission, Swap, Gross Profit, Net Profit.

ByBit

ByBit → Orders → Closed P&L → Export (top right). Required columns: Contract, Side, Average Entry Price, Average Exit Price, Closed P&L, Qty, Final time.
